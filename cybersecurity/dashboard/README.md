# Dashboard

Kibana dashboards for real-time threat visualization.

## Import Kibana Saved Objects
1. Open Kibana → Stack Management → Saved Objects
2. Import `kibana_dashboard.ndjson`

## Key Dashboards
- **Threat Overview** — live alert feed, severity distribution
- **Network Monitor** — traffic anomaly timeline
- **UEBA** — user behavior deviation chart
- **Agent Health** — per-agent event throughput
